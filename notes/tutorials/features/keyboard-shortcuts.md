## üí° Tips

### Platform Compatibility
- **Windows/Linux**: Use <kbd>Ctrl</kbd> key for shortcuts
- **Mac**: Use <kbd>‚åò</kbd> (Command) key instead of <kbd>Ctrl</kbd>

### Customizable Shortcuts
You can customize some keyboard shortcuts in **Settings** ‚Üí **Advanced** ‚Üí **Keyboard Shortcuts**. The following shortcuts can be customized:
- New tab
- Search  
- Settings
- Tag filtering
- Bookmark note

### Browser Compatibility
Some shortcuts may conflict with browser defaults. The wiki provides alternative shortcuts (using <kbd>Alt</kbd>) to avoid conflicts.
