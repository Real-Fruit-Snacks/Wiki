## 🚀 **Deploy in 60 Seconds**

<details open>
<summary><strong>📦 Option 1: GitHub Pages (Recommended)</strong></summary>

```bash
# 1. Fork this repository (click Fork button above)
# 2. Go to Settings → Pages → Source: GitHub Actions
# 3. Your wiki is live at: https://[username].github.io/Wiki/
```

That's it. Seriously. ✨

</details>

<details>
<summary><strong>🦊 Option 2: GitLab Pages</strong></summary>

```bash
# Clone the repository
git clone https://github.com/Real-Fruit-Snacks/Wiki.git
cd Wiki

# Push to GitLab
git remote add gitlab https://gitlab.com/your-username/wiki.git
git push gitlab main

# Enable GitLab Pages in project settings
# Live at: https://your-username.gitlab.io/wiki/
```

</details>

<details>
<summary><strong>💻 Option 3: Run Locally</strong></summary>

```bash
# Clone and serve
git clone https://github.com/Real-Fruit-Snacks/Wiki.git
cd Wiki
python3 -m http.server 8000

# Open http://localhost:8000
```

</details>

<details>
<summary><strong>📥 Option 4: Download Release Package</strong></summary>

1. [Download Latest Release](https://github.com/Real-Fruit-Snacks/Wiki/releases/latest)
2. Extract `notes-wiki-v3.6.5-complete.zip`
3. Upload to any web hosting or open `index.html` locally
4. Complete offline package with all 150 themes included!

</details>

## 📖 **Writing Your First Note**

Create a markdown file in `/notes/`:

```markdown
---
title: My First Note
tags: [getting-started, tutorial]
author: Your Name
created: 2025-06-22
status: published
category: tutorial
aliases: [first-note, getting-started-guide]
keywords: [tutorial, getting-started, first, note]
---

# My First Note

Welcome to your new knowledge base! 

**Features to try:**
- Press `?` to see keyboard shortcuts
- Press `Ctrl+K` to search
- Right-click this note to bookmark it
- Press `F` for focus mode
- Press `Ctrl+Shift+S` for Quick Notes panel
```

Then rebuild the search index:
```bash
python3 build.py
```
